ESP32 Telegram bot with relay:
- Controls a relay: `/click`, `/clicklong`, `/clicklength <ms>`
- Shows IP info: `/ip`
- Sends uptime: `/uptime`
- Pings devices: `/pingpc`, `/pingip <ip>`
- Flashes onboard LED: `/flash`
- Sends status on startup + every 12h
- Auto WiFi reconnect
- `/help` shows commands
